# BCTF — Bitcoin Community Trust Fund

A long-term, community-owned Bitcoin treasury governed by constitution and code.

---

## What is BCTF?

BCTF is an attempt to build a financial institution that could still function, preserve capital, and accumulate Bitcoin twenty years from now.

It is not a yield protocol. It is not a trading fund. It is not optimised for hype.

It is a Bitcoin treasury with a written constitution, an autonomous governor, and a governance model where influence is earned through contribution — not capital alone.

---

## Core Principles

- Bitcoin is the primary reserve asset
- No leverage. Ever.
- The treasury cannot be fully emptied by any proposal
- All wallets are public and auditable at all times
- Constitutional amendments require a mandatory delay period
- Governance influence reflects sustained contribution, not only capital
